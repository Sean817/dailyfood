## 🐛 如果遇到问题

1. **页面无法加载**
   - 确认开发服务器运行：检查终端是否有 `npm run dev` 的输出
   - 确认地址正确：http://localhost:5173

2. **搜索没有结果**
   - 尝试输入：鸡蛋、米饭、菠菜、苹果、牛奶
   - 这些食物都在数据库中

3. **营养计算为0**
   - 确认食物名称与数据库完全匹配
   - 确认输入了数量

4. **数据丢失**
   - 检查浏览器控制台是否有错误
   - 确认浏览器允许 localStorage
